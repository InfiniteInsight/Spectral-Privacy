# Broker Definitions

This directory contains TOML definition files for data brokers that Spectral can interact with.

## Schema

See `schema.toml` for the complete schema documentation.

## Contributing

When adding a new broker definition:

1. Copy an existing definition as a template
2. Research the broker's opt-out process
3. Fill in all required fields
4. Test the definition manually
5. Submit a PR with the `broker-definition` label
